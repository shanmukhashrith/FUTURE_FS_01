# Personal Professional Portfolio (FUTURE_FS_01)

## Project Overview
This repository contains the source code for my personal professional portfolio, developed as Task 1 for the Future Interns Full Stack Web Development Internship. 
The portfolio features a custom "Modern Tactical" dark theme utilizing glassmorphism UI principles, custom 3D CSS animations, and dynamic DOM manipulation for scroll-reveal effects. 

## Features
* **Responsive Design:** 100% mobile-friendly layout across all viewports.
* **Interactive UI:** Scroll-reveal animations, 3D tilt-effect project cards, and a custom high-tech cursor.
* **Modern Aesthetics:** Dark theme with tactical cyan/gold accents and volumetric background glows.

## Technologies Used
* **HTML5:** Semantic structuring
* **CSS3:** Custom variables, keyframe animations, 3D transforms
* **JavaScript:** Intersection Observers, DOM manipulation, asynchronous text effects

## Setup & Deployment Steps

### Local Setup
1. Clone this repository to your local machine using:
   `git clone https://github.com/shanmukhashrith/FUTURE_FS_01.git`
2. Navigate into the project directory.
3. Open the `index.html` file in any modern web browser to run the project locally. No build tools or package managers are required.
